libcommon Documentation

libcommon est une bibliothèque regroupant un certain nombre de types et fonctions de base très utiles en C.

Elle gère par exemple:

  • Un jeu de test unitaire
  • Des raccourcis pour réaliser des cast
  • Des types basiques
  • Des fonctions de hashage basiques

Types basiques:

  • String
  • Vector
  • List
  • Map

Fonctionalités avancées:

  • autoString() (gestion automatique de pointeurs de chaines de caractères)
