void JSONParser_setExitOnException ( bool  value)

Permet de définir si le programme doit quitter ou non.

Par défaut, en cas d'exception, le programme s'arrêtera. Sinon, l'exception sera gérée par libcjson et continuera. Les fonctions ayant provoquée une exception retourneront 0 a la place d'un quelconque pointeur.

Paramètres
valuetrue/1 pour quitter en cas d'exception, false/0 sinon.
